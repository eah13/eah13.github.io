What is Clojure?

To varying degrees of resolution: * a programing language * a LISP dialect * Clojure is a dynamic programming language that targets the Java Virtual Machine (and the CLR, and JavaScript). It is designed to be a general-purpose language, combining the approachability and interactive development of a scripting language with an efficient and robust infrastructure for multithreaded programming. Clojure is a compiled language - it compiles directly to JVM bytecode, yet remains completely dynamic. Every feature supported by Clojure is supported at runtime. Clojure provides easy access to the Java frameworks, with optional type hints and type inference, to ensure that calls to Java can avoid reflection.  (from Clojure.org)

Trying out Clojure with no install

Trying out Clojure is super easy, thanks to the geniuses who wrote 4Clojure.com.
  • Open a REPL in one tab (tryclj.com is a good one)
  • Open one of the Clojure Problems in another tab
  • 4Clojure problems are fill-in-the-blank code.  The REPL will help you test pieces of code.
  • Submit - If you got a green light, congrats!  Move on!  If not, repeat from the top
  • (Optional) Create an account to track your progress and play Code Golf
Note there is no compiling needed here.  You are running a real Clojure REPL in the browser.
